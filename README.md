# songfarmer-site

Marketing website for [Songfarmer](https://songfarmer.com) — a songwriting method, book, and app by Owen Temple & Gordy Quist.

Built with [Hugo](https://gohugo.io/). Hosted on [Netlify](https://www.netlify.com/).

## Quick Start

```bash
# Install Hugo (macOS)
brew install hugo

# Run local dev server
hugo server

# Build for production
hugo --minify
```

## Adding a Blog Post

```bash
hugo new blog/my-post-title.md
```
